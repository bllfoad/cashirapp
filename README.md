### Technologies Used

**Frontend:**
- Next.js
- Tailwind CSS
- TypeScript
- Shadcn
- Radix UI

**Backend:**
- AWS
- AWS Storage
- PostgreSQL

### Features

- State Persistence on Refresh
- Caching with Redis
- Stock Quantity Control
- Mobile Responsiveness
- Admin Dashboard
- Frontend Error Handling
- Download Receipt
- Dark Mode
